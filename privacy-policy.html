GG Buddy — Privacy Policy

Last updated: February 28, 2026

Overview

GG Buddy is a Chrome extension that helps you compare game prices across stores using the GG.deals API. We are committed to protecting your privacy.

Data Collection

We do not collect, store, or transmit any personal information.

The extension operates entirely locally in your browser. All data remains on your device or, optionally, in your Chrome sync storage (which is managed by Google).

Data Stored Locally

  • Wishlist: Game IDs and titles you choose to track
  • Preferences: Theme, accent color, notification settings
  • Price Cache: Temporary cached price data to reduce API calls
  • Price History: Historical price records for games you've looked up
  • Recent Searches: Your recent search queries

Chrome Sync

If enabled, your wishlist and preferences may be synced across devices using Chrome's built-in sync storage. This is managed by Google's Chrome Sync infrastructure and subject to Google's privacy policy.

External API Requests

The extension makes requests to the GG.deals API to fetch game prices, search results, and bundle information. These requests include:

  • Game Steam IDs or search queries
  • Your selected region/currency preference

No personal information, browsing history, or user identifiers are sent with these requests.

Permissions Explained

  • storage: Save your wishlist, preferences, and cache locally
  • activeTab: Detect games on the current store page when you click the extension
  • notifications: Send price drop alerts (optional, user-controlled)
  • alarms: Schedule periodic price checks (optional, user-controlled)
  • contextMenus: Add "Look up on GG.deals" right-click option

Third-Party Services

This extension uses the GG.deals API. Please review GG.deals' Privacy Policy for their data practices.

Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected in the "Last updated" date above.
